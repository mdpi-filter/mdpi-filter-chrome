About Notandia

Notandia, previously distributed as MDPI Filter, adds publisher context and explainable post-publication signals while you read scholarly literature.

Notandia is an independent open-source project. It is not affiliated with, authorized by, or endorsed by MDPI AG, Crossref, Retraction Watch, NCBI, browser vendors, or any publisher or data provider.

Features

  • Highlights or hides MDPI results on supported scholarly-search pages.
  • Identifies MDPI references and related numerical citations where page structure permits.
  • Optionally checks DOI metadata for formal retractions, concerns, corrections, reinstatements, withdrawals, and duplicate-publication findings.
  • Shows chronology, provenance, coverage, deferred checks, and unresolved identifiers without assigning a hidden quality score.
  • Keeps integrity lookups off by default and supports a zero-network configuration.
